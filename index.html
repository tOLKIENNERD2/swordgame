Controls

Warrior

  • A - Move Left
  • D - Move Right
  • W - Jump
  • S - Attack
  • Spacebar - Block

Archer

  • A - Move Left
  • D - Move Right
  • W - Jump
  • Q - Aim Up
  • S - Aim Down
  • Spacebar - Shoot Arrow

General

  • T - Toggle Player Type (before game start)
  • Enter - Start Game
  • R - Restart (when game over)

Tips

  • Perform combos by attacking quickly
  • Block to reduce incoming damage
  • Collect power-ups for advantages

Game Options

Player Type: warrior
